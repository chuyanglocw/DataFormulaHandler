### ***版本:0.1***
### ***环境:Python***
### ***作者:ChuYangLOCW***
## 介绍
***这个项目旨在使用公式处理较多数据.***

## 用法
* ***创建一个txt文件***
``` txt
<variables>
a b
</variables>
<py>
def print_area(area):
	print(f"Area:{area}")
</py>
<formulas>
a+b
area=a * b
f-print_area(area)
</formulas>
<data>
;Rect1
1 2
;Rect2
2 4.0
</data>
```
#### 解释
* >\<variables>  **开始输入变量.**
* >\</variables> **停止输入变量.**
* >\<py> **开始执行脚本.**
* >\</py> **停止执行脚本.**
* >\<formulas>  **开始输入公式.**
* >\</formulas> **停止输入公式.**
* >\<data>  **开始处理数据.**
* >\</data> **停止处理数据.**
* >***f-*** **是标识不用显示返回值的公式.**
* >***;*** **是数据分区的标识.**
